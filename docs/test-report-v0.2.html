功能概览

🔄 Redis 注册中心

  • Agent 自注册（TTL + 心跳续约）
  • 按技能索引查询（Redis SET）
  • Pub/Sub 实时上下线通知
  • TTL 过期自动注销
  • health_sweep 清理孤儿数据

🔌 Agent 适配器

  • OpenAI API 适配器（/v1/chat/completions）
  • CLI 适配器（子进程 stdin/stdout）
  • MCP 适配器（JSON-RPC 2.0）
  • AdapterManager 路由工厂
  • 统一 AgentBackend 接口

🛡 Circuit Breaker

  • 三状态机：CLOSED → OPEN → HALF_OPEN
  • 滑动窗口失败率检测
  • 慢调用检测（slow_call_threshold）
  • 自动恢复探测
  • 手动 reset 强制恢复

🌐 Gateway API

  • POST /api/v1/agents/register
  • POST /api/v1/agents/{id}/heartbeat
  • POST /api/v1/agents/{id}/deregister
  • GET /api/v1/agents?skill=xxx
  • POST /api/v1/agents/{id}/invoke

📦 swarm-sdk

  • AgentClient 异步 HTTP 客户端
  • async context manager 自动注销
  • start_heartbeat_loop 后台心跳
  • register / heartbeat / deregister

⚙️ 配置扩展

  • RedisConfig（url, ttl, interval）
  • main.py 生命周期集成
  • Gateway router 自动挂载
  • Redis 连接失败优雅降级
